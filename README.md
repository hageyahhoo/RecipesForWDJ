# RecipesForWDJ
Recipes of Chef for wdj.


## How to set up Serverspec

### 1) Use Gemfile & Bundler
	$ bundle install

### 2) Use gem install
	$ gem install test-kitchen
	$ gem install kitchen-vagrant
	$ gem install serverspec



## Tips
